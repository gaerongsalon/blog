# @blog/utils

General-purpose utilities shared by workspace packages.

## Key Paths

- `lib/deadlineTimer.ts`: Deadline helper for retry loops.
- `lib/sleep.ts`: Promise-based timeout helper.
- `lib/sortUniqueStrings.ts`: Deduplicates and locale-sorts string arrays.

## Usage

Keep this package limited to small, dependency-light helpers that are broadly reusable.
